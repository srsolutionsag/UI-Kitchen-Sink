All UI-Components in the UI-Kitchen-Sink MUST have an assigned state, which MUST be one of the following:

State: Implemented for X.Y

This component has been implemented for ILIAS version X.Y.

State: Accepted

This component has been accepted by the JF but is not yet implemented.

State: Proposal

This component has been accepted by the JF but is not yet implemented.

State: To be revised

The component has been cataloged from ILIAS 5.1 but has yet to be revised.
