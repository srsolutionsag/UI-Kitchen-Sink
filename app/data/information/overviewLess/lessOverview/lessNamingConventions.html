If proposing a new variable, you MUST consider the following points:

  • Less variables MUST be named [il-component-they-are-primarely-used-in-functionality-css-attribute-name]. Example: il-modal-body-bg-color.
  • All less variables MUST be stored in the variables.less file.
  • In the variables.less file, the variables MUST be structured into sections. Each section MUST start with the following name Tag [//== Name of the Section]
  • The section names must match the categories the components are stored in. All variables concerning colors must be stored in the section [//== Colors] as an example.
  • The variable must be described in the according less category of the UI-Kitchen-Sink.

If new less classes are needed they MUST be named according the following pattern [il-id-of-component-name]. Example: il-modal-title
