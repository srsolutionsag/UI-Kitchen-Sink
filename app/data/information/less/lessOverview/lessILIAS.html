Since ILIAS 5.0 Bootstrap 3 is used as clientside framework. With the Design Revision 5.1 the JF decided that Bootstrap Concepts SHOULD be used where possible. One important element of Bootstrap 3 is the usage of less.

All new CSS-Styles of Modules or Services in ILIAS MUST be written in less, stored templates/default/less/Modules (or templates/default/less/Services). Please consult less.org to make use of the advantages of less. Before adding new less, the following steps listed in the UI-Kitche-Sink workflow MUST be performed.
